## requirements

* libxml2
* libxml++2
* libevent
* libvorbis
* libogg
* tup (for compilation)
* python2.x (for bindings and some misc utils)
* pycrypto (for bindings and some misc utils)

## installation

There is no installation script at this time, so you have to
```sh
$ tup init
$ tup upd
```
then copy appropriate files where you want them to be.
